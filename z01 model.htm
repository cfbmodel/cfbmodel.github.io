Rather than create ratings from scratch, or even start with my ratings from the previous season, the guiding principle of this model is to start with the final Kenneth Massey Computer Power Ratings of the previous season, and then apply changes for the upcoming season. For this reason, I sometimes refer to it as a "delta" model.

The first step is to identify the data elements that I want to apply. These are reviewed and adjusted every season based on relevance, availability, and accuracy. Elements considered include:

  • Historical ratings (measured by year-over-year change or regression to the mean)
  • Previous season injury luck (not often available)
  • Previous season fumble recovery luck
  • Coaching staff attributes and results
  • Roster talent
  • Unit rankings
  • Quarterback ratings
  • Returning starters
  • Returning production (yards, tackles, etc)
  • Recruiting
  • Transfers
  • Age
  • Experience
  • Phase continuity
  • Preseason all-conference teams

Next, a series of correlations are computed for each element (see table below), and the highest one for each element is selected. I try to use as much historical data as possible.

In most cases, these are the steps followed:

(1) Scale the element to the range of the final Kenneth Massey Computer Power Ratings of the previous season. For example, if the highest correlation corresponds to line 5 in the table, and if the element Rating Values range from 10 to 30, and if the Massey Rating Values range from 40 to 100, then an element of 20 would become 70.
(2) Repeat for the same element in the previous season, then subtract it.
(3) Multiply by the appropriate correlation factor(s). If the element has significant overlap with another element, potentially apply a weighting factor to minimize duplication.
(4) Add to the cumulative new ratings.

Once all elements have been processed, the cumulative new ratings are scaled to the Massey range in order to properly constrain the new range.
